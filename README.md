# Devindo Generator

This package will generate folder and file

```bash
depedency_need_to_remove.dart
app
    - router.dart
    - locator.dart
    - thid_party_app
assets
screens
services
utils
```

## How to use

Open the command palette (macOS: Shift+Command+P, Windows: Ctrl+Shift+P) and type, "Generate Devindo."

```bash
flutter pub run build_runner build
```
